# AGENTS.md (作業の絶対ルール)
※トークン節約のため、本ルールを厳守の上、簡潔に動作すること。

## 1. 開発環境と技術スタック
- 静的HTML/CSS/JSのみで構成。Webアプリ化は不可。
- React, Next.js, TypeScript, npm, ビルド, 開発サーバーは絶対に使用しない。
- 動作確認はローカルサーバーを立てず、`index.html` をブラウザで直接開く。

## 2. Mac環境の保護（厳禁事項）
Mac本体やグローバル環境の変更は絶対禁止。
- 禁止: アプリ導入、Homebrew、npm/npx、pip/仮想環境、sudo、PATH/シェル設定変更、外部ネットワークアクセス。

## 4. 許可なく実行してよい操作
ターミナルを使わず、エディタのファイル操作機能でプロジェクトフォルダ内を直接操作することのみ許可。
- HTML, CSS, JS, SVG, 画像, robots.txt, site.webmanifest 等の作成・編集
- assets, css, js, images 等の子フォルダ作成
